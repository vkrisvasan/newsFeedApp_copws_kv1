# newsFeedApp_copws_kv1

## Description
GenAI News is a web application that brings you the latest news about Generative AI (GenAI) in different tiles based on their popularity. The news articles are fetched from a news API and displayed in a grid layout.

## Setup and Run
1. Clone the repository:
    ```bash
    git clone https://github.com/githubnext/newsFeedApp_copws_kv1.git
    ```
2. Navigate to the project directory:
    ```bash
    cd newsFeedApp_copws_kv1
    ```
3. Open `index.html` in your preferred web browser.
